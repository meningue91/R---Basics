## First Analyses
This study analyses usage data downloaded via API to reach an econometric conclusion. To achieve this, the data were cleaned, organized, and used to generate descriptive statistics and graphics using basic procedures in R.

### Data
The data was donwload from World Banking server by API in R as a xml file and represents the period from 2000 to 2022. After that, the data was turned into RData file and arrange to produced descriptive statistics. Moreover, the indicator choosed were Gini Index, GDP per capita, and population. The missing data were filled by linear imputation, and spline imputation. The cleaned applyed used basic tools from R as dplyr and Tidyverse.
